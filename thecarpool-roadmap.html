Pre-Ride AI Call & Assistant

Twilio + Claude API + ElevenLabs TTS

TheCarPool bypasses easy-to-ignore push alerts with automated voice confirmation calls 1 hour before departure, dynamically logging answers to coordinate schedules.

1
Queued Call Triggers Server checks ride list at T-60min and launches call webhook through Twilio.
2
Claude Intent Classification Processes spoken user feedback (e.g. "I need 10 extra minutes") into functional parameters.
3
ElevenLabs TTS Response Generates natural voice replies in localized Hindi-English accents.
